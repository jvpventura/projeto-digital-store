# 👟 Digital Store - E-commerce Front-end

Este é o projeto final do módulo de Front-end da Digital College. O objetivo deste projeto é construir a interface de uma loja virtual (Drip Store) utilizando tecnologias modernas, seguindo um design system pré-estabelecido no Figma.

## 🌐 Projeto ao Vivo (Deploy)

Acesse a aplicação rodando em produção:
👉 **[Clique aqui para acessar a Digital Store](https://SEU_LINK_DA_VERCEL_AQUI.vercel.app)**

## 🚀 Tecnologias Utilizadas

- **React.js** (com Vite)
- **Tailwind CSS** (para estilização rápida e responsiva)
- **React Router Dom** (para navegação entre páginas)
- **Lucide React** (para ícones)

## 📋 Funcionalidades Implementadas

- ✅ Layout fiel ao Figma (Home, Lista de Produtos e Detalhes do Produto).
- ✅ Navegação dinâmica usando rotas (SPA).
- ✅ Galeria de imagens interativa na página do produto.
- ✅ Seletores dinâmicos de tamanho e cor.
- ✅ Filtragem de produtos por marca e texto (Barra de Pesquisa).
- ✅ Ordenação de produtos por preço (Maior/Menor).
- ✅ Renderização condicional de preços com desconto.

## 🛠️ Como rodar o projeto localmente

Siga os passos abaixo para executar o projeto na sua máquina:

1. Clone este repositório:
   bash
   git clone (https://github.com/jvpventura/projeto-digital-store)
2. Acesse a pasta do projeto:
   bash
   cd drip-store

3. Instale as dependências:
   bash
   npm install

4. Inicie o servidor de desenvolvimento:
   bash
   npm run dev

5. Abra o seu navegador no link fornecido no terminal (geralmente http://localhost:5173/).
